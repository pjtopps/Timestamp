Timestamp Microservice

Pass a string as a parameter.
If this string is a unix time or a date, lo and behold, an object shalt be returned to you!
This object will contain the corresponding unix timestamp AND natural date!!

Eg.

https://fcc-timestamp-pjt.glitch.me/December%2015,%202015
or
https://fcc-timestamp-pjt.glitch.me/1450137600000

...will both return the following:

{"unix":1450137600000,"natural":"December 15, 2015"}
